# GradientForge

Neural networks you can see. Train on your Mac, visualize what the network learned, and export to Core ML in one click — no code required.

## Overview

GradientForge is a macOS application that makes neural networks accessible and understandable. Train neural networks directly on your Mac, watch them learn in real-time with interactive visualizations, and seamlessly export your trained models to Core ML format.

## Features

- 🧠 **Easy Neural Network Training** - Train models on your Mac without writing any code
- 👁️ **Real-time Visualization** - See exactly what your neural network is learning
- ⚡ **One-Click Export** - Export trained models to Core ML format instantly
- 🎯 **Intuitive Interface** - User-friendly design that makes machine learning accessible to everyone
- 💻 **Native macOS Integration** - Built for Mac with native performance

## Requirements

- macOS 10.15 or later
- Mac with Apple Silicon or Intel processor

## Getting Started

### Installation

1. Download the latest release from the [Releases](https://github.com/chickweed01/GradientForge/releases) page
2. Drag the app to your Applications folder
3. Launch GradientForge

### Basic Usage

1. **Prepare Your Data** - Import your training data
2. **Configure Your Network** - Set up your neural network architecture using the visual builder
3. **Train** - Click train and watch your network learn in real-time
4. **Visualize** - Explore what your network learned through interactive visualizations
5. **Export** - Export your trained model to Core ML with one click
